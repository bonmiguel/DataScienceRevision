# Pandas Workspace

This repository is a small workspace for learning Pandas in Python.

## Structure

- `src/` holds runnable Python scripts.
- `docs/` holds notes, observations, and short writeups.
- `requirements.txt` lists the Python packages used in the workspace.
- `.venv/` is the local virtual environment used by VS Code and the terminal.

## Getting Started

Activate the environment and install the listed dependencies:

```bash
source .venv/bin/activate
pip install -r requirements.txt
```

Run the starter script with:

```bash
python src/main.py
```

## Working Style

1. Keep small experiments and scripts in `src/`.
2. Keep explanations and progress notes in `docs/`.
3. Add or update dependencies in `requirements.txt` when needed.
4. Use `.venv/` as the local environment for this workspace.
